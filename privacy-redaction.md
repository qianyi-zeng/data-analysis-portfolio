# 脱敏与公开说明

本作品集基于真实工作经历进行方法论整理，但所有内容均已脱敏处理。

## 不公开的内容

- 原始 PDF、Excel、PPT、图片、截图。
- 内部知识库链接、BI 链接、系统链接。
- 商家名称、主播名称、用户 ID、店铺 ID、商品 ID、视频 ID。
- 预算池、资金池、审批群、内部流程编号。
- 具体资源位量级、补贴规则、平台策略细节。
- 可反推出公司经营数据的 GMV、订单、ROI、NPS 明细。
- 个人手机号、私人邮箱、微信号。

## 保留的内容

- 问题拆解方法。
- 指标体系和分析框架。
- 报表自动化结构。
- 周报复盘模板。
- 脱敏 SQL 写法。
- 可公开的个人成果描述。

## 原则

1. 能说明能力，不暴露数据。
2. 能描述方法，不复刻内部流程。
3. 能展示结构，不上传原始材料。
4. 能讲业务价值，不泄露公司机密。
