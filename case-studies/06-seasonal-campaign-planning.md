# 案例 6：季节性专项活动立项与预算策略

## 背景

某头部内容电商平台的食品家居行业在大促结束后，会进入约两个月的夏季长周期经营窗口。这个阶段既要承接大促余温，又要覆盖暑期、三伏、开学前等消费场景，涉及生鲜食品、家居家纺、清洁收纳、数码家电等多个类目。

单纯做促销会带来两个问题：一是目标和预算容易只围绕 GMV 讨论，忽视新客、复购和钱效；二是活动玩法容易分散，直播、短视频、商城、搜索等场域缺少统一的指标和资源约束。因此需要一套从机会判断、目标测算、预算分配到过程监控的立项框架。

## 目标

- 将季节性消费机会拆成可招商、可配置、可监控的活动方案。
- 在 GMV 冲刺之外，同时约束新客质量、复购、预算消耗和补贴 ROI。
- 根据上一期活动复盘结果，提前设计止损规则和预算动态调整机制。
- 建立商家准入、商品准入和反作弊规则，避免低质供给消耗补贴。

## 我的工作

1. 拆解夏季消费场景：把用户需求拆成消暑清凉、开学焕新、家居换季等主线，并匹配对应类目、货盘和目标人群。
2. 设计活动节奏：区分预热蓄水期和成交爆发期，让短视频种草、直播专场、限时补贴和重点货盘上新形成节奏配合。
3. 测算目标和预算：基于前期活动表现、类目结构、渠道贡献和用户规模，拆分日均 GMV、用户数、复购率、预算占比和 ROI 约束。
4. 做上一期复盘归因：分析 GMV 未达、短视频流量波动、低价拉新品类复购弱、时令生鲜品质波动等问题，并转化为本期策略假设。
5. 设计分层玩法：针对头部主播、中腰部主播、新商和短视频作者分别设置不同激励方式，避免资源平均分配。
6. 建立风险控制：把体验分、品退、差评、NPS、价格校验、补贴率、异常用户和黑产风险纳入活动准入与过程监控。

## 立项框架

| 模块 | 关键问题 | 方法 |
| --- | --- | --- |
| 机会判断 | 这期活动为什么值得做 | 节点需求、季节货盘、历史活动复盘、类目增长空间 |
| 目标测算 | 目标是否可达 | 对比基期、拆日均目标、拆用户结构、看复购和 ROI 底线 |
| 预算分配 | 钱应该投给谁 | 按类目、渠道、玩法和阶段拆预算，设置动态调整机制 |
| 货盘策略 | 什么商品能承接流量 | 时令生鲜、消暑食品、清凉家电、收纳清洁、开学用品 |
| 渠道玩法 | 如何拉动交易 | 直播排位赛、拉新直播间、短视频话题、精准券补 |
| 风险治理 | 如何避免无效增长 | 商家准入、商品准入、价格校验、补贴互斥、反作弊监控 |
| 复盘闭环 | 下期如何优化 | 按 GMV、用户、复购、ROI、体验和渠道波动复盘 |

## 关键分析

### 1. 从上一期问题反推本期策略

上一期活动不是简单“目标没完成”，而是由多个因素共同影响：

- 短视频流量分发策略调整后，低客单内容曝光承压，影响家居类目新客获取。
- 拉新品类偏低频，虽然能带来新客，但复购弱，导致整体新客质量不足。
- 时令生鲜有天然复购潜力，但受品质波动、品退要求和开播稳定性影响，部分高复购货盘没有完全放量。
- 大促临近会产生用户消费延迟，需要把活动定位从单次冲量转为大促后承接和长周期经营。

因此本期策略不是简单加预算，而是把预算投向更能同时拉 GMV、拉新客、带复购的场景。

### 2. 活动目标同时看结果和质量

活动立项目标不能只写 GMV，需要同时约束：

- 结果指标：支付 GMV、订单量、日均目标、目标完成率。
- 用户指标：去重用户数、新客、老客、新客 GMV、新客复购。
- 费用指标：预算消耗、补贴率、ROI、单用户成本。
- 渠道指标：直播、短视频、商城、搜索等场域贡献。
- 体验指标：NPS、品退率、差评率、商家服务能力。

这能避免活动短期靠高补贴冲 GMV，但后续复购、体验和钱效变差。

### 3. 预算采用动态分配

活动期预算按阶段、类目和玩法拆分，但不一次性固定到底。过程监控中根据目标达成情况动态调整：

- 达成目标的类目和商家，可以进入更高预算档位。
- 未达目标但仍有转化效率的供给，观察后小额补贴。
- ROI、体验或复购低于底线的供给，及时降档或暂停。
- 对高撬动率商家做定向配置，把资源集中到更有承接能力的供给。

## 业务价值

- 把季节性活动从“活动执行表”提升为“经营策略方案”，能说明为什么做、投给谁、怎么监控和何时止损。
- 将 GMV、用户增长、复购、补贴 ROI 和体验治理放到同一个决策框架里。
- 帮助运营团队在直播、短视频、商城、搜索等多场域之间做资源取舍。
- 通过上一期复盘反推本期策略，形成活动立项 -> 过程监控 -> 复盘优化的闭环。
