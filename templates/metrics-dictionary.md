# 指标字典模板

这是一份脱敏后的电商经营分析指标字典，可用于日报、周报、看板和专项分析。

## 结果指标

| 指标 | 定义 | 常用拆分 |
| --- | --- | --- |
| 支付 GMV | 用户完成支付的交易金额 | 行业、类目、渠道、商家、日期 |
| 成交 GMV | 含补贴或业务定义后的成交金额 | 类目、渠道、活动 |
| 订单量 | 符合统计口径的订单数 | 新老客、渠道、商家 |
| QTD 达成率 | 季度累计完成 / 季度目标 | 行业、类目、负责人 |
| 剩余日均 Gap | 剩余目标 / 剩余天数 - 当前日均 | 行业、渠道 |

## 流量与转化

| 指标 | 定义 | 用途 |
| --- | --- | --- |
| 曝光 | 内容或直播间被展示次数 | 判断流量规模 |
| 点击 | 用户点击进入次数 | 判断素材吸引力 |
| CTR | 点击 / 曝光 | 判断流量承接第一步 |
| CVR | 成交用户或订单 / 点击或看播 | 判断交易转化 |
| GPM | GMV / 曝光 * 1000 | 判断流量变现效率 |
| OPM | 订单量 / 曝光 * 1000 | 判断订单转化效率 |

## 供给指标

| 指标 | 定义 | 用途 |
| --- | --- | --- |
| 有效开播 | 达到有效时长或质量标准的开播数 | 判断直播供给 |
| 开播时长 | 主播或商家开播总时长 | 判断承接能力 |
| 发文量 | 短视频发布数量 | 判断内容供给 |
| 动销视频数 | 产生成交的视频数 | 判断内容质量 |
| 优质新商 | 符合冷启标准的新供给 | 判断增长后备供给 |

## 用户指标

| 指标 | 定义 | 用途 |
| --- | --- | --- |
| 新客数 | 首次购买或业务定义新用户 | 判断拉新能力 |
| 老客数 | 历史购买用户 | 判断存量运营 |
| 复购率 | 观察期内再次购买用户 / 首购用户 | 判断长期价值 |
| 新客 GMV 占比 | 新客 GMV / 总 GMV | 判断增长结构 |

## 费用与体验

| 指标 | 定义 | 用途 |
| --- | --- | --- |
| 补贴金额 | 平台或活动承担的优惠成本 | 判断费用规模 |
| 补贴率 | 补贴金额 / 含补贴成交金额 | 判断补贴强度 |
| ROI | 增量收益 / 费用成本 | 判断钱效 |
| NPS | 推荐用户占比 - 贬损用户占比 | 判断体验健康度 |
| 品退率 | 品质相关退款或退货占比 | 判断商品风险 |
| 差评率 | 差评订单或评价占比 | 判断服务风险 |
