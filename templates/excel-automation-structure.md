# Excel 自动化报表结构

这是一套适合业务中台日报/周报的 Excel 结构设计。公开版本只展示结构和公式思路，不包含原始数据。

## 工作簿结构

| Sheet | 用途 |
| --- | --- |
| 参数页 | 日期、季度起止、目标、筛选条件 |
| 日报数据 | 面向业务阅读的结果汇总 |
| 周报数据 | 周维度同比、环比、QTD、Gap |
| GMV 底表 | 从 BI 或数仓导出的明细数据 |
| 流量底表 | 曝光、点击、看播、播放等过程数据 |
| 供给底表 | 开播、发文、动销、新商等供给数据 |
| 用户底表 | 新老客、复购、核销等用户数据 |
| 校验页 | 总量核对、缺失值检查、异常值检查 |

## 自动化思路

1. 用参数页控制所有日期区间，避免手工改多个公式。
2. 底表只负责承接数据，不做复杂展示。
3. 展示页用 SUMIFS、COUNTIFS、XLOOKUP、IFERROR 等公式自动汇总。
4. 同比、环比、目标达成率和剩余 Gap 统一公式口径。
5. 校验页检查总量是否与底表一致、关键字段是否缺失、指标是否出现异常跳变。

## 常用公式

```excel
+=SUMIFS(GMV底表[支付GMV], GMV底表[日期], $B$2, GMV底表[渠道], C$1)
```

```excel
+=IFERROR(本周日均 / 上周日均 - 1, "")
```

```excel
+=IFERROR(QTD累计 / 季度目标, "")
```

```excel
+=IFERROR((季度目标 - QTD累计) / 剩余天数 - 当前日均, "")
```

## 设计原则

- 原始数据、计算逻辑、展示结果分离。
- 让运营同学只需要替换底表或刷新数据源。
- 用校验页减少人工检查。
- 所有口径变化必须记录在变更日志中。
